A library for working with popular HTTP/S APIs in Rust.

** Status
*EXPERIMENTAL*
- most things don't work out-of-box, if at all
- NWS, ipapi, and SMTPS are most functional (see =demo=)

** Features
- Clients
   - [X] NWS (US Weather)
   - [X] ipapi (public IP lookup)
   - [ ] openai (beta API)
   - [ ] Google Cloud
   - [ ] AWS
   - [X] SMTPS (smtp.gmail.com)
- [ ] Authentication
   - [ ] JWT
   - [ ] OAuth
